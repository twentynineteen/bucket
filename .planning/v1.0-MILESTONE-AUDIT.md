3. **Dynamic import convention deviation** — `useBakerTrelloIntegration.ts:53,70`: `await import('@features/Baker')` instead of static barrel import. Static `vi.mock` won't intercept.

4. **TrelloCard type name collision** — Both `@features/Baker` and `@features/Trello` export `TrelloCard` with incompatible shapes. Currently handled with aliased imports.
